<h3>Docker instructions</h3>
pulling images for your local machine:

```javascript
  $ docker pull yarins2000/petshopapp:1.0
  $ docker pull yarins2000/petshopdb:1.0
```

running the containers of these images:

```javascript
  $ docker-compose up -d
```
